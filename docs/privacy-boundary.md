# Privacy Boundary

This public repository is designed for employer review and portfolio proof only.

## Public

- General checkout UX architecture and compatibility notes.
- Public-safe descriptions of preview-first rollout and data-normalization patterns.
- Sanitized snippets that do not expose live checkout behavior.

## Private

- Production source code.
- Payment/provider details, customer data, checkout logs, store-specific settings, and private UX tests.
- Internal conversion rules or operational checkout procedures.

## Rule

Public samples should explain the engineering approach without exposing live checkout operations.
