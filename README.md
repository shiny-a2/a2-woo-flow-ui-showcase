# A2 Woo Flow UI Showcase

Public-safe showcase for a private Persian WooCommerce cart, checkout, OTP, and thank-you UI compatibility layer.

This repository is not a source mirror. Production source, checkout configuration, customer data, payment/provider details, and store-specific UX rules stay private.

## What This Demonstrates

- Checkout/cart UX engineering for Persian WooCommerce stores.
- Preview-first UI rollout thinking for sensitive purchase flows.
- Digit normalization, location data handoff, and compatibility boundaries.
- Separation between settings, frontend behavior, and checkout safety.
- Public-safe documentation of conversion-flow improvements without exposing production internals.

## Tech Stack

- WordPress plugin architecture
- WooCommerce checkout/cart UI
- PHP
- CSS/JavaScript frontend layer
- Persian UX and compatibility patterns
