# Sample Code

This directory contains Phase 3 public-safe snippets. The samples demonstrate checkout UI workflow-control approach without mirroring production source code.

## Published Samples

- `php/checkout-field-policy.php` — generic checkout field requirement and validation policy.
- `php/mobile-verification-state.php` — simplified mobile verification state without provider details.
- `js/digit-normalization-ui.js` — localized digit normalization for frontend checkout inputs.

## Sample Rules

- Use fictional field names, sample phone numbers, and generic UI copy.
- Keep snippets short enough for review.
- Explain what each snippet demonstrates.
- Exclude payment details, SMS credentials, customer data, order IDs, live checkout settings, page IDs, coupon data, provider names, internal action names, nonce names, option names, meta keys, and production UI copy.
- Do not include production-ready checkout code or live rollout commands.

## Status

Phase 3 is active with two PHP snippets and one JavaScript snippet. These files are intentionally simplified and should be read as review samples, not as production checkout code.
